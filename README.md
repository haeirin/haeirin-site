# My Shop - 쇼핑몰 프로젝트

## 프로젝트 개요
소규모 온라인 쇼핑몰 정적 웹사이트. 상품 조회, 장바구니, 주문/결제(무통장), 회원가입/로그인, 관리자 대시보드를 제공합니다.

---

## 현재 완료된 기능

### 👤 회원 시스템
- **닉네임 규칙**: 영문(a-z, A-Z)만 허용, 2~12자 (한글·숫자 입력 시 자동 제거)
- **로그인**: 닉네임 + 전화번호 (비밀번호 없음, 전화번호가 비밀번호 역할)
- **회원가입**: 3단계 (기본정보 → 연락처/주소 → 약관동의)
  - 이메일 필수 (계정 찾기에 사용)
  - 전화번호가 로그인 비밀번호로 자동 설정
  - 카카오 주소 검색 API 연동
  - 닉네임/이메일/전화번호 중복 체크
- **계정 찾기** (`find-account.html`)
  - 닉네임 찾기: 이름 + 이메일로 조회
  - 전화번호 변경: 닉네임 + 이메일 본인 확인 후 새 전화번호로 변경
- **관리자 로그인**: 로그인 페이지 하단 "관리자 페이지 접속" 버튼 → 닉네임+비밀번호 입력 (기본값: admin / admin1234)

### 🛍️ 상품 시스템
- **상품 목록** (`index.html`): 카테고리 필터, 검색, 페이지네이션
- **상품 상세** (`product.html`): 색상·사이즈 옵션 선택, 재고 확인, **유통기한 표시**, 장바구니·바로구매
- **재고 자동 차감**: 주문 완료 시 stock 자동 감소 (옵션별 포함)
- **재고 복구**: 관리자가 주문 취소 시 재고 자동 복구
- **유통기한 관리**: 관리자 페이지에서 상품별 유통기한 설정 가능 (날짜 선택 또는 직접 입력)

### 🛒 장바구니 & 주문
- **장바구니** (`cart.html`): 선택 삭제, 수량 변경, 무료배송 기준 표시
- **주문/결제** (`checkout.html`): 카카오 주소 검색, 계좌이체, 배송 메모
- **주문 완료** (`order-complete.html`): 입금 확인 버튼, 주문 상세 확인

### 🔧 관리자 (`admin.html`)
- **상품 관리**: 등록·수정·삭제, **JPG 파일 업로드** (drag & drop, base64 변환), **유통기한 설정**
- **주문 관리**: 입금 확인·배송 처리·완료·취소, 상태 변경 시 대시보드 즉시 갱신
- **재고 관리**: 개별 재고 편집, 옵션별 재고 설정
- **회원 관리**: 회원 목록 조회
- **설정**: 쇼핑몰 이름·설명, 배송비·무료배송 기준, 계좌 정보
- **회사 정보**: 사업자 정보, 주소, 연락처, SNS 등록
- **택배 업로드**: 결제완료 주문 엑셀 다운로드 (택배사별 양식, 우편번호·주소·상세주소 분리)
- **비밀번호 변경**: 관리자 비밀번호 변경 (현재 비밀번호 확인 후 변경)

### 📦 택배 업로드 (`delivery-upload.html`)
- **직접 입력**: 수신자 정보 직접 입력 후 택배사 양식으로 엑셀 다운로드
- **주문 가져오기**: 결제완료 주문을 자동으로 불러와 택배 양식 생성
- **카카오 주소 검색**: 🔍 주소검색 버튼으로 우편번호·도로명주소 자동 입력
- **3칸 분리 입력**: 우편번호 / 주소(도로명) / 상세주소 각각 독립 셀로 관리
- **지원 택배사**: CJ대한통운(일반/대량), 한진택배(일반/대량/대량10↑), 롯데택배(일반/대량), 오늘픽업(대량)
- **박스타입**: X, S, A, B, C, D, E, F (택배사별 허용 타입 자동 필터링)
- **SheetJS** 활용 엑셀(.xlsx) 파일 생성

### 🔑 관리자 비밀번호 재설정 (`admin-reset.html`)
- 비밀번호를 잊어버린 경우 사용하는 긴급 재설정 페이지
- 닉네임 + 새 비밀번호 입력으로 재설정
- 관리자 계정 목록 조회, 삭제, 새 계정 생성 기능

### 👤 마이페이지 (`mypage.html`)
- **주문 내역**: 주문 카드·상태 바·상세 모달
- **내 정보 수정**: 휴대폰·주소 수정 (카카오 주소 검색 연동)
- **비밀번호 변경**: 현재 비밀번호 확인 후 새 비밀번호 설정

## 로그인 방법

### 일반 회원
- 닉네임(영문 2~12자) + 가입 시 등록한 전화번호

### 관리자
1. `login.html` 하단 **"⚙️ 관리자 페이지 접속"** 버튼 클릭
2. 닉네임: `haeirin` / 비밀번호: `haeirin1368`

## 데이터 모델

### users
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| nickname | text | 영문 2~12자 (로그인 ID) |
| name | text | 실명 |
| email | text | 이메일 (계정 찾기용) |
| password | text | hashPassword(전화번호 숫자) |
| phone | text | 전화번호 (로그인 비밀번호) |
| zipcode | text | 우편번호 |
| address | text | 기본 주소 |
| address_detail | text | 상세 주소 |
| privacy_agreed | bool | 개인정보 동의 |
| role | text | user / admin |

### products
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| name | text | 상품명 |
| description | text | 상품 설명 |
| price | number | 판매가 |
| category | text | 카테고리 |
| images | array | 이미지(base64 또는 URL) |
| thumbnail | text | 대표 이미지 |
| expiry_date | text | 유통기한 (YYYY-MM-DD 또는 텍스트) |
| colors | array | 색상 옵션 |
| sizes | array | 사이즈 옵션 |
| stock | number | 총 재고 |
| stock_by_option | text | 옵션별 재고 (JSON) |
| is_active | bool | 판매 활성화 여부 |

### orders
주문번호, 회원 정보, 상품 목록, 결제금액, 배송비, 결제 방법/상태, 배송 정보 포함

### cart
사용자별 장바구니 상품 목록

### settings
쇼핑몰 설정 (배송비, 계좌정보, 쇼핑몰 이름 등)

## 이미지 업로드 규칙
- **관리자 상품 등록**: JPG 파일만 허용 (accept="image/jpeg,.jpg,.jpeg")
- 최대 파일 크기: 5MB
- 업로드 방식: 파일 선택 또는 드래그 앤 드롭 → base64 변환 후 DB 저장
- URL 직접 입력 보조 지원

✏️ 직접 입력 방법

  1. 원하는 택배사를 선택합니다
  2. 각 행에 이름 · 핸드폰 · 우편번호 · 주소(도로명) · 상세주소 · 박스타입을 입력합니다
  3. 🔍 주소검색 버튼을 클릭하면 카카오 우편번호 서비스가 열려 우편번호·도로명주소가 자동 입력됩니다
  4. [엑셀 다운로드] 버튼으로 해당 택배사 양식 파일을 생성합니다

📋 주문에서 가져오기

  1. [주문에서 가져오기] 탭 선택
  2. [주문 목록 불러오기] 클릭 → 결제완료 주문 조회
  3. 배송할 주문을 체크하고 택배사와 박스타입을 선택
  4. [선택한 주문 가져오기] 클릭 → 우편번호·주소·상세주소 각 셀 확인·수정
  5. [엑셀 다운로드] 클릭
